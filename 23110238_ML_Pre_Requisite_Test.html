Question-1: How many multiplications and additions do you need to perform a matrix multiplication between a (n, k) and (k, m) matrix? Explain.

Answer: n x m x k multiplications & n x m x (k-1) additions

Explaination: Our Resulting Matrix will be (n,m), therefore having n*m elements. For each row x column, we need to perform k multiplications and k-1 additions. This operation will be done to get each element of the resultant matrix, therefore, there would be n x m x k multiplications and n x m x (k-1) additions.

  def __init__(self, years_list):

    list_years = self.years

def broadcasting_sum(a,b):
  ax = len(a[0])
  ay = len(a)
  bx = len(b[0])
  by = len(b)
  if ax == bx and ay == by:
    return a+b
  elif ax == bx and ay>by:
    b1 = b
    for i in range(ay-by):
      b = np.concatenate((b,b1), axis=0)
    return a+b

  elif ax == bx and ay<by:
    a1 = a
    for i in range(by-ay):
      a = np.concatenate((a,a1), axis=0)
    return a+b
  elif ax > bx and ay==by:
    b1 = b
    for i in range(ax-bx):
      b = np.concatenate((b,b1), axis=1)
    return a+b
  elif ax < bx and ay==by:
    a1 = a
    for i in range(bx-ax):
      a = np.concatenate((a,a1), axis=1)
    return a+b

print(broadcasting_sum(a,b))

def argmin(y):
  min_i = 0
  min_j = 0
  n_1 = len(y)
  n_2 = len(y[0])
  for i in range(n_1):
    for j in range(n_2):
      if y[i][j] < y[min_i][min_j]:
        min_i = i
        min_j = j
  return min_i, min_j
